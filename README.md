Overview
SEO Rank Tracker is a MERN stack-based web application that allows users to keep track of their keywords' rankings and their websites' SEO performances. Users can put their target keyword and domain name within the application to know about the ranking of the website on Google search engine results pages (SERPs). All ranking information like positions, page number, titles, and snippets are captured and displayed by the application.

Website SEO analysis is another feature provided by the web app. In order to get an SEO report, users just have to enter the domain in the web application. This will give them a comprehensive SEO report with on-page SEO parameters such as page title, meta descriptions, headings, image alt tag, canonical link, internal links, and external links. An SEO score is generated for each parameter along with recommendations for improvement.

For the backend of the application, technologies such as Node.js, Express.js, and MongoDB have been used, while React.js has been utilized for creating a frontend UI of the application. SerpAPI and Cheerio libraries are used for acquiring search results from Google search engines and website scraping and SEO data extraction respectively.

### Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Mongoose
- SerpAPI
